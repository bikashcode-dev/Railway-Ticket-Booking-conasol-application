# Railway Ticket Booking Console Application

## Overview

This is a console-based Railway Ticket Booking System developed using Core Java. The project simulates basic railway reservation functionalities such as ticket booking, cancellation, train management, and passenger handling.

## Tech Stack

* Core Java
* Collection Framework
* JSON
* OOP Concepts

## Features

* User Login & Registration
* Train Search
* Ticket Booking & Cancellation
* Seat Availability Check
* Passenger Management
* Console-Based Menu System

## Collections Used

* ArrayList
* HashMap
* List
* Map

## How to Run

```bash
git clone https://github.com/bikashcode-dev/Railway-Ticket-Booking-conasol-application.git
```

Open the project in IntelliJ IDEA or any Java IDE and run the `Main.java` file.
