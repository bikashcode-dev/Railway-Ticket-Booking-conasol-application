# Railway Ticket Booking Console Application

## Overview

This is a console-based Railway Ticket Booking System developed using Core Java. The project simulates real-world railway reservation functionalities such as ticket booking, cancellation, train management, passenger handling, and secure user authentication.

The application follows Object-Oriented Programming principles and uses Java Collections Framework for efficient data management. User credentials are securely handled using password encoding techniques, and the project is designed in a scalable way so it can later be integrated with SQL databases like MySQL or PostgreSQL.
